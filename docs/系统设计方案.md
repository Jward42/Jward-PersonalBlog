3. 内容集合 Schema (src/content/config.ts)这是我们管理 Markdown 的核心。AI 助手需要创建 src/content/config.ts 并定义以下 collections：import { defineCollection, z } from 'astro:content';

// 1. 博客 (Blog) 集合
const blogCollection = defineCollection({
  type: 'content', // 意味着是 .md 文件
  schema: z.object({
    title: z.string(),
    description: z.string(),
    publishDate: z.date(),
    tags: z.array(z.string()),
    heroImage: z.string().url().optional(), // 封面图 (来自 Cloudinary)
    isDraft: z.boolean().default(false),
  }),
});

// 2. 摄影 (Photography) 集合
// 注意：图片本身在 Cloudinary, .md 文件只用于存储元数据
const photographyCollection = defineCollection({
  type: 'data', // 意味着是 .json 或 .yaml, 但我们用 .md (content) 更好管理
  type: 'content',
  schema: z.object({
    title: z.string(),
    description: z.string().optional(),
    captureDate: z.date(),
    tags: z.array(z.string()),
    imageUrl: z.string().url(), // !! 关键：来自 Cloudinary 的图片 URL
    aspectRatio: z.number().default(1), // 可选：用于优化瀑布流布局
  }),
});

export const collections = {
  'blog': blogCollection,
  'photography': photographyCollection,
};
4. 核心组件实现指南src/layouts/BaseLayout.astro:必须包含 <BaseHead />, <Header />, <body> 和 <Footer />。<body> 中应包含一个 <slot />，用于注入页面内容。应导入 global.css。src/pages/blog/[slug].astro:这是博客详情页，使用 getStaticPaths 来生成页面。必须使用 post.render() 来渲染 Markdown 内容。渲染的内容 必须 包含在一个 <article class="prose lg:prose-xl"> 标签内，以激活 @tailwindcss/typography 插件的美化效果。src/pages/photography/index.astro:使用 getCollection('photography') 获取所有照片数据。应使用 CSS Grid (非 Flexbox) 来实现瀑布流或网格布局，例如：grid-cols-2 md:grid-cols-3 lg:grid-cols-4。Cloudinary 图片处理：在 .md 文件中，heroImage 和 imageUrl 字段将是我们手动粘贴的 Cloudinary URL。在渲染 <img> 标签时，AI 助手应主动使用 Cloudinary 的 URL 转换 API 来请求优化后的图片（例如，添加 w_800,q_auto,f_auto 参数）以提高性能。5. 开发执行阶段 (Step-by-Step)AI 助手请按以下顺序协助我：阶段 1: 项目初始化执行 npm create astro@latest (或同等命令) 初始化项目。添加 Tailwind CSS (@tailwindcss/astro)。配置 tailwind.config.mjs，特别是要引入 @tailwindcss/typography 插件。阶段 2: 基础布局创建 src/layouts/BaseLayout.astro。创建 src/components/Header.astro (包含 Home, Blog, Photography, About 链接)。创建 src/components/Footer.astro (包含版权信息)。创建 src/assets/global.css (包含 Tailwind base 导入)。阶段 3: 内容与 Schema创建 src/content/config.ts 并填入上述 Schema。创建 2-3 个 示例 blog .md 文件 (使用占位符图片 URL)。创建 2-3 个 示例 photography .md 文件 (使用占位符图片 URL)。阶段 4: 页面实现 (主页)构建 src/pages/index.astro。实现“个人简介”部分。实现“最新博客” (Hooks)：使用 getCollection('blog')，排序并取前 3 篇。实现“精选摄影” (Hooks)：使用 getCollection('photography')，取前 6 张。阶段 5: 页面实现 (博客与摄影)构建 src/pages/blog/index.astro (博客列表页)。构建 src/pages/blog/[slug].astro (博客详情页，重点关注 prose class)。构建 src/pages/photography/index.astro (摄影画廊页)。
